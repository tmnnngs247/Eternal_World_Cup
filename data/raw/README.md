# Raw data

Place the source datasets here when rebuilding the project locally:

- `players_15.csv` … `players_22.csv`
- `player_23.csv`
- `player_24.csv`
- `players_25.csv`
- `players_26.csv`
- `fifa_fbref_merged.csv`
- `WorldCupPlayers.csv`
- `WorldCupMatches.csv`
- `WorldCups.csv`
- `world_cup_rosters_2026.json` optional, from the pasted roster metadata

Large raw files are intentionally not required by the Streamlit app. The app reads from `data/processed/`.
